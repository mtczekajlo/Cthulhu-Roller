# Cthulhu Roller

**Cthulhu Roller** is a Call of Cthulhu RPG 7E dice roller bot for Discord.

## Installation

[shuttle.rs quick start](<https://docs.shuttle.rs/getting-started/quick-start>)

## Usage

```text
/roll 2d4+4
```
